# Node.js C++ Addon: Hello World with WinAPI

This is a simple demonstration of a native Node.js addon written in C++. It uses Windows API (WinAPI) to retrieve the computer name and returns it to JavaScript.

## 🚀 What this proves
- I can write **C++ code** that integrates with Windows API.
- I can compile it into a **native Node.js module**.
- I understand **node-addon-api** and **node-gyp** build system.
